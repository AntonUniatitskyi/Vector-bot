# 🛡 Vector Bot

Telegram-бот для автоматизации контента авторского канала про кибербезопасность,
OSINT и полезные консольные утилиты. Работает по принципу **Human-in-the-Loop**:
бот сам ищет и генерирует черновики постов, а публикует их только админ —
вручную, по кнопке.

## Возможности

- **💡 Своя идея** — даёшь тему, бот ищет инфу в интернете (DuckDuckGo) и
  генерирует пост (Gemini) в авторском стиле, с живым стримингом текста
  прямо в чат (как у ChatGPT).
- **🎲 Придумай сам** — бот сам предлагает тему, не повторяя то, что уже
  публиковалось.
- **🛰 Автопилот** — раз в N часов (по умолчанию 3) проверяет RSS-ленты
  (BleepingComputer, The Hacker News, Хабр) и GitHub (тренды по темам
  `security-tools`, `osint`), присылает черновики по новым находкам.
- **Модерация в один клик**: ✅ Опубликовать в канал / ✏️ Переписать /
  🗑 Отклонить.
- Доступ строго для одного администратора — все чужие апдейты игнорируются.

## Стек

Python 3.11+ · aiogram 3 · SQLite + SQLAlchemy (async) · APScheduler ·
Gemini API (`google-genai`) · DuckDuckGo Search (`ddgs`) · Docker

## Быстрый старт (локально)

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

cp .env.example .env
# заполни .env своими значениями — см. таблицу ниже

python vector_bot.py
```

## Быстрый старт (Docker)

```bash
cp .env.example .env
# заполни .env своими значениями

docker compose up -d --build
docker compose logs -f   # смотреть логи
```

База данных при этом лежит в `./data/vector.db` на хосте (volume) — переживает
пересоздание контейнера.

## Переменные окружения (`.env`)

| Переменная | Обязательна | Описание |
|---|---|---|
| `BOT_TOKEN` | ✅ | Токен от [@BotFather](https://t.me/BotFather) |
| `ADMIN_ID` | ✅ | Твой числовой Telegram ID (узнать у [@userinfobot](https://t.me/userinfobot)) |
| `CHANNEL_ID` | ✅ | Канал для публикации: `@username` или числовой ID (`-100...`). Бот должен быть добавлен туда админом |
| `GEMINI_API_KEY` | ✅ | Ключ из [Google AI Studio](https://aistudio.google.com/app/apikey) — бесплатно |
| `GEMINI_MODEL` | — | По умолчанию `gemini-2.5-flash` |
| `DB_URL` | — | Путь к SQLite. Для Docker переопределяется автоматически в `docker-compose.yml` |
| `SEARCH_REGION` | — | Регион DuckDuckGo (`wt-wt` = без региона) |
| `BLOCKED_DOMAINS` | — | Домены, которые не предлагать как источники, через запятую (`ru`) |
| `AUTOPILOT_INTERVAL_HOURS` | — | Как часто автопилот проверяет источники (по умолчанию 3) |
| `AUTOPILOT_MAX_ITEMS` | — | Максимум черновиков за один прогон (по умолчанию 3) |

## Команды бота

| Команда / кнопка | Действие |
|---|---|
| `/start` | Приветствие + клавиатура |
| `/idea [текст]` | Сгенерировать пост по теме |
| 💡 Своя идея | То же самое, но через диалог |
| 🎲 Придумай сам | Бот сам предлагает тему |
| `/check_now` | Запустить проверку источников автопилота вне расписания |
| `/sources` | Список источников автопилота с id и статусом |
| `/togglesource [id]` | Включить/выключить источник |

## Структура проекта

```
vector_bot.py     — точка входа, хендлеры, FSM, планировщик
db.py             — модели SQLAlchemy (posts, sources) и работа с БД
llm.py            — генерация текста через Gemini (обычная и потоковая)
search.py         — поиск через DuckDuckGo с фильтром доменов
autopilot.py       — фоновый мониторинг RSS/GitHub по расписанию
ui.py             — общие UI-хелперы (клавиатура модерации, безопасная отправка)
Dockerfile / docker-compose.yml — контейнеризация
```

## Как это устроено (коротко)

```
Telegram → Router → AdminProtectFilter → Handler
                                             │
                    ┌────────────────────────┼────────────────────────┐
                    ▼                        ▼                        ▼
              search.py                  llm.py                   db.py
           (DuckDuckGo,               (Gemini: генерация,       (SQLite: очередь
          без .ru доменов)         стрим, переписывание)        и история постов)
```

Черновик всегда проходит через модерацию в личке админа — бот никогда не
публикует в канал самостоятельно.

## Известные ограничения

- Бесплатный тариф Gemini Flash: ~10 запросов/мин, 250/день — для одного
  администратора этого достаточно, но не гоняй `/idea` слишком часто подряд.
- Стриминг (`SendMessageDraft`) работает только в личных чатах — это и есть
  единственный сценарий использования в этом боте.
- SQLite подходит для одного инстанса бота. Если когда-нибудь понадобится
  несколько реплик — потребуется PostgreSQL.
